Парсер
-----------

**Режимы работы парсера**

    1. Просмотреть все жалобы.
    2. Посмотреть не закрытые жалобы.
    3. Выключить парсер.
    4. Посмотреть до определенного момента.
    5. Посмотреть срок решения от.
    6. Обновить сегодняшние жалобы.
    7. Обновление браузера.
    8. Получить скриншот.
    9.
    10. Парсинг авторов
